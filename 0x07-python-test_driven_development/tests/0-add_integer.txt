How to use  0-add_integer.py
============================

This module defines an integer addition function ``add_integer(a, b=98)``.

Usage
=====

``add_integer(...)``` function returns the value of addition of two integers.

::
    >>> add_integer = __import__("0-add_integer").add_integer
    >>> add_integer(2, 3)
    5

it also add negative values
::
    >>> add_integer(2, -3)
    -1
    >>> add_integer(-3, -5)
    -8

float number treated as int and only the int value is added
::
    >>> add_integer(2.2, 2.4)
    4
    >>> add_integer(2.2, .4)
    2
    >>> add_integer(-0.2, -2.9)
    -2

float and int can be combined
::
    >>> add_integer(2.3, 6)
    8

if b or a is not int it raise a ValueError say value must be ant int
::
    >>> add_integer("a", 1)
    Traceback (most recent call last):
        ...
    TypeError: a must be an integer
    >>> add_integer(1, "hello")
    Traceback (most recent call last):
        ...
    TypeError: b must be an integer

if both of args are integers a type message print for the first argument only
::
    >>> add_integer("h", "g")
    Traceback (most recent call last):
        ...
    TypeError: a must be an integer
