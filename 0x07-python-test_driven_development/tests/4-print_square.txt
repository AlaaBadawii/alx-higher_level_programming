=============================
this is 4-print_square module
=============================

this module explains how to use ``print_square(size)`` function

Usage:
------

``print_square(...)``` function prints a square with the char #.

::
    >>> print_square = __import__("4-print_square").print_square
    >>> print_square(4)
    ####
    ####
    ####
    ####

if size == 0 it print a empty line
::
    >>> print_square(0)
    <BLANKLINE>

::
    >>> print_square(-3)
    Traceback (most recent call last):
        ...
    ValueError: size must be >= 0

::
    >>> print_square("h")
    Traceback (most recent call last):
        ...
    TypeError: size must be an integer
